This dashboard offers a quick and easy-to-understand summary of the E-Commerce store's financial and operational performance. It is designed to track high-level Key Performance Indicators (KPIs) and identify the top drivers of sales across products, customers, and regions.

## Dashboard Key Metrics & Objectives

The primary goal of this report is to monitor the core financial health of the E-Commerce business.

- **Total Sales:** **438K**  
- **Total Profit:** **37K**  
- **Total Quantity:** **5,615**  

## Core Analysis Areas

| Section | Description | Key Visuals |
|--------|-------------|-------------|
| **Product Performance** | Analysis of **Total Sales by Category** and a breakdown of **Total Sales and Profit by Sub-Category**. | Donut Chart, Stacked Column Chart |
| **Customer & Geography** | Identification of the **Top 5 Customers by Total Sales** and the contribution of the **Top States** and **Top Cities** to overall revenue. | Bar Charts, Donut Chart |
| **Payment Analytics** | Breakdown of **Total Sales by Payment Mode** (COD, UPI, Credit Card, Debit Card) to understand customer preference. | Donut Chart |
| **Filters** | Includes slicers for filtering the data by **Quarter** and **Month** for flexible time-based analysis. | Slicer Filters |

## Data Sources

- E-Commerce Transactional and Sales Order Data  
- Customer and Product Master Data  

## Technical Details

This dashboard was developed using **Power BI Desktop**. It provides a high-density, single-page view designed for leadership, utilizing bright colors and clear visual separation for rapid data comprehension.
