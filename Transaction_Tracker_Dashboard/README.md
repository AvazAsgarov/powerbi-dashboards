This dashboard provides a comprehensive view of transaction data, sales performance, customer demographics, and order fulfillment status. It is designed to offer actionable insights into product category performance, customer engagement, and operational efficiency.

## Dashboard Key Metrics & Objectives

The primary goal of this dashboard is to track and analyze the key components of the business transaction cycle.

- **Customer Overview:** **86.8K** Customers  
- **Total Sales:** **995T** Total Sales  
- **Customer Feedback:** **3.16** Average Feedback Rating  

## Core Analysis Areas

| Section | Description | Key Visuals |
|--------|-------------|-------------|
| **Sales Performance** | Breakdown of sales volume across major **Product Categories** and **Brands**. | Treemap for Sales Distribution |
| **Customer Insights** | Analysis of **Customer Age vs. Income** and correlation with **Amount Spent**. Includes **Average Ratings** by customer segments (Premium, New, Regular). | Scatter Plot, Bar Chart |
| **Operational Tracking** | Monitors order lifecycle across **Delivered**, **Shipped**, **Processing**, and **Pending** statuses. | Bar Chart, Donut Chart |

## Data Sources

- Sales and Transaction Log Data  
- Customer Demographic and Feedback Data  

## Technical Details

This dashboard was developed using **Power BI Desktop**.  
Data was transformed and modeled using **Power Query (M Language)** and **DAX**.
